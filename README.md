# EventFuse
An online tool that helps film creators and editors to schedule and orgainse their events in a new way.

## Some Features

* [ ] ⌨️ Login/signup
* [ ] 📝 Add events
* [ ] 🙈 View all events
* [ ] 🔍 Search
* [ ] ➡️ Assign user to an event 
* [ ] 🗃 Show events for specific user
* [ ] 💬 Chat room

## Technology Stack

* Client Side:
  * HTML
  * CSS
  * JavaScript
  * jQuery
  * React/Vue
  * Materialize
    
* Server Side:
  * NodeJS
  * HandlebarJS
  * Express
  * MongoDB
